# 🌤️ Modern Weather Dashboard

A sleek, responsive Weather Dashboard built with a focus on **Glassmorphism** design. This project was created as part of my **6th-semester** exploration into front-end web development and AI-driven interfaces.

## 🚀 Features
- **Glassmorphism UI:** Modern, semi-transparent background with backdrop-blur effects.
- **Real-time Search:** Search for any city to see current weather conditions.
- **Responsive Design:** Works beautifully on both Desktop and Mobile.
- **Clean Code:** Structured with HTML5, CSS3, and Vanilla JavaScript.

## 🛠️ Tech Stack
- **HTML5** (Structure)
- **CSS3** (Styling & Layout)
- **JavaScript** (Logic)
